About the Hackathon

With this year's theme being around OPEN SOURCE & AI, this hackathon will focus on building an open-source project that leverages platforms like DigitalOcean's Gradient AI Platform (along with any tech stack of your choice) to solve an interesting real-world problem or demonstrate a novel ML/AI experience in the space of open source.

Participation Guidelines

  • Open to individuals and teams (max 3 people)
  • Projects must be public GitHub repositories and open-source (choose a permissive license like MIT, Apache-2.0, or CC0)
  • Participants must submit their GitHub repo URL via the submission form

Submission Requirements

Each submission must include the following files in your GitHub repository:

README.md

Project summary, live demo link (if applicable), setup + run steps, expected inputs/outputs

LICENSE

e.g., MIT or Apache-2.0

CONTRIBUTING.md

Short guide to run, test, and contribute

CODE_OF_CONDUCT.md

Standard contributor code of conduct

Environment Setup

requirements.txt / environment.yml or Dockerfile for reproducible environment

Demo (Optional)

demo.gif or short video link (highly recommended)

Judging Criteria

Your projects will be judged by a short 10 minute presentation that you will showcase after the hacking period is over. The judges will score you based on these metrics:

  • Use of AI platform ‚Äî How effectively you leverage the AI platform capabilities
  • Completeness of the solution ‚Äî How well-rounded and finished your project is
  • Impact ‚Äî The potential real-world impact and usefulness of your solution
  • UI and UX ‚Äî User experience and interface design quality
